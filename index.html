首次使用需要设置

在 GitHub 创建一个访问令牌，就能在这里直接发布了。

  1. 打开 github.com/settings/tokens
  2. 点 Generate new token (classic)
  3. Note 填 watery-site
  4. Expiration 选 No expiration
  5. 勾选 repo（一整栏）
  6. 点 Generate token
  7. 复制生成的令牌（ghp_ 开头）
